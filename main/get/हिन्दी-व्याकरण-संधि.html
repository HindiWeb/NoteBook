संधि का व्युत्पति-

यह दो शब्दों के योग से बना है। सम् + धि = ‘सम्’ का अर्थ होता है, ‘पूर्णतया’ और ‘धि’ का अर्थ होता है, ‘मिलना’ अथार्त दो ध्वनियों या वर्णों का पूर्णतया मिलना संधि कहलाता है।

परिभाषा- दो ध्वनियों या वर्णों के परस्पर मेल से उत्पन्न ध्वनि-विकार को संधि कहते है।

उदाहरण-

षट् + आनन = षडानन

निः + छल = निश्छल (विकार उत्पन्न होना ही संधि है।)

उदहारण-

राधा + कृष्ण = राधाकृष्ण (यहाँ कोई भी विकार उत्पन्न नही हुआ अतः यह संधि नहीं  ‘संयोग’ होगा।)

संधि के तीन भेद होते है- स्वर संधि, व्यंजन संधि, विसर्ग संधि।

स्वर संधि:

प्रथम वर्ण में हलन्त् हो, तब वह वर्ण अपने ही पंक्ति के पाँचवें वर्ण में बदल जाएगा।

उदाहरण:

वाक् + मय = वाङ्मय (विश्व की समस्त लिखित साहित्य वाङ्मय कहलाता है)

सत् + न्यासी = संन्यासी

उत् + नायक = उन्नायक

विद्वत + मंडल + विद्वन्मंडल

सत् + मार्ग = सन्मार्ग

उत् + नति = उन्नति

चित् + मय = चिन्मय  

संधि विच्छेद करना:

उन्नति = उत् + नति

चिन्मात्रा = चित् + मात्रा

वाङ्मय = वाक् + मय

उन्नायक = उत् + नायक

चिन्मय – चित् + मय

विसर्ग संधि

परिभाषा- विसर्ग के साथ ‘स्वर’ अथवा ‘व्यंजन’ के मेल से उत्पन्न ध्वनि-विकार को विसर्ग संधि कहते हैं।

उदाहरण-

निः + आशा = निराशा (यहाँ विसर्ग और स्वर का मेल है)

निः + चिंता = निश्चिंत (यहाँ विसर्ग और व्यंजन का मेल है)

यहाँ चाहे ‘स्वर’ आए या ‘व्यंजन’ आए। चेंज (बदलाव) सिर्फ विसर्ग में ही होगा, स्वर या व्यंजन में कोई भी परिवर्तन नहीं होगा।

विसर्ग (:) + स्वर / व्यंजन हो बदलाव सिर्फ विसर्ग में होगा। 

विसर्ग (:) सिर्फ इनमे बदलेगा – श् ष् स् र् ई ऊ ओ, अपरिवर्तित होगा या लोप हो जाएगा।

नियम 1- विसर्ग का श् में बदलना:

विसर्ग (:) के बाद च / छ / श = विसर्ग ‘श्’ में बदल जाएगा

उदाहरण:

निः + चय = निश्चय 

पुनः + च = पुनश्च

पुरः + चरण = पुरश्चरण (अनुष्ठान)

निः + चिंता = निश्चिंत

मनः + चिकित्सक = मनश्चिकित्सक

निः + छल = निश्छल

निः + शब्द =निश्शब्द

अतः + चेतना = अंतश्चेतना

यश: + शरीर = यशश्शरीर

निः + शुल्क = निश्शुल्क

निः + छल = निश्छल

दु: + शासन = दुश्शासन

संधि विच्छेद करना

‘श्’ के बाद यदि च, छ, श आ जाए तो ‘श्’ विसर्ग में बदल जाएगा। (हलंत का मतलब है आधा)

उदाहरण:

उदाहरण:

चतुष्टीका = चतु: + टीका = चतुष्टीका (चारों वेदों की व्याख्या चतुष्टीका कहते है)

निष्कंटक = निः + कंटक

नियम 3 – विसर्ग का ‘स्’ में बदलना इसमें दो नियम है:

(क) विसर्ग के बाद में ‘त’‘थ’ या ‘स’ आ जाए तो विसर्ग ‘स्’ में बदल जाता है।

उदाहरण:

विसर्ग के बाद अगर  कर / कार / वर / कृत / पति / पर आ जाए तो विसर्ग ‘स्’ में बदल जाएगा।

पुरः + कर्ता = पुरस्कर्ता

परः + पर = परस्पर

तिरः + कार = तिरस्कार

पुरः + कृत = पुरस्कृत

संधि विच्छेद:

स् के बाद कर / कार / कृत / पति / पर / वर आ जाए तो ‘स्’ विसर्ग में बदल जाएगा।

उदहारण:

नमस्कार = नमः + कार

वनस्पति = वनः + पति

6. विसर्ग का ‘ई’ की मात्रा में बदलना:

7. विसर्ग का ‘ऊ’ में बदला:

उदाहरण:

दु: + राज = दूराज (बूरा है जो शासन)

8. विसर्ग का अपरिवर्तित रहना:

प्रातः + काल = प्रातःकाल

मनः + कामना = मनःकामना

अधः + पतन = अधःपतन

अतः + एव = अतएव
